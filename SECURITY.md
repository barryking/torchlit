# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.2.x   | :white_check_mark: |
| < 0.2   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in Torchlit, please report it responsibly via [GitHub Security Advisories](https://github.com/barryking/torchlit/security/advisories/new).

Please **do not** open a public issue for security vulnerabilities.

We will acknowledge receipt within 48 hours and aim to release a fix within 7 days for confirmed issues.
